# nvir-streamlit-demo

This is a demo repo to demonstrate nilvana vision inference runtime with streamlit.

## Get Started

```python
python3 -m venv .venv
source .venv/bin/activate
pip3 install -r requirements.txt

streamlit run app.py
```
